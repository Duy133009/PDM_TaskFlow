# InsightPM Platform

A modern, comprehensive project management solution designed to streamline workflow, enhance team collaboration, and boost productivity. InsightPM combines powerful features like Kanban boards, Gantt charts, and real-time analytics into a single, intuitive interface.

![InsightPM Dashboard](https://images.unsplash.com/photo-1460925895917-afdab827c52f?auto=format&fit=crop&q=80&w=2426&ixlib=rb-4.0.3)

## üöÄ Features

-   **Interactive Dashboard**: Get a high-level overview of project health, task distribution, and team velocity with real-time charts and metrics.
-   **Kanban Board**: Visualize workflow and manage tasks effortlessly with drag-and-drop functionality.
-   **Gantt Chart**: Plan and schedule projects effectively with a timeline view of tasks and dependencies.
-   **Resource Management**: Optimize team allocation and track workload to prevent burnout.
-   **Advanced Analytics**: Gain deep insights into project performance, efficiency, and time utilization.
-   **Time Tracking**: integrated time logging to monitor effort and improve estimation accuracy.
-   **Real-time Collaboration**: Seamlessly collaborate with team members with instant updates.

## üì¶ Installation

1.  **Clone the repository**

    ```bash
    git clone https://github.com/yourusername/insightpm-platform.git
    cd insightpm-platform
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

3.  **Configure Environment Variables**

    Create a `.env` file in the root directory and add your Supabase credentials:

    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Start the development server**

    ```bash
    npm run dev
    ```
